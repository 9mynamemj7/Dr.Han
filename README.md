## 나무의사 2차시험을 위한 테스트 웹 프로젝트

### 목표

> <b>나무의사 시험을 준비하는 사람을 대상으로 시험준비를 도와주는 웹을 개발한다.</b></br>
1차 목표 - 테스트 기능 하나로 배포, 친구들에게 뿌려보기 </br>
2차 목표 - 세가지 기능을 가지고 나무의사 준비생들에게 뿌려 수요 파악</br>
3차 목표 - 나도 커피값 받을래
